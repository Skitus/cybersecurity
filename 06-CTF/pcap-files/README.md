# PCAP Files

Training network captures used for Wireshark analysis practice.

## How to Download

All files are saved as Claude artifacts in this conversation:
https://claude.ai/chat/463ce71f-d6e0-4cc9-ba82-747739ba2d1c

Open the link, click "Download all" in the top right of the Artifacts panel, then move the files here.

## Files

| File | Category | What to Look For |
|---|---|---|
| voip-ctf.pcapng | VoIP Analysis | SIP/RTP traffic, call recording extraction |
| malware-ctf.pcapng | Malware Traffic | C2 beacons, suspicious DNS, data exfil |
| llmnr-practice.pcapng | LLMNR Poisoning | LLMNR/NBT-NS requests, credential capture |
| attack-ctf.pcapng | Attack Traffic | Port scans, exploit attempts |
| webattack-ctf.pcapng | Web Attacks | SQLi, XSS, directory traversal in HTTP |
| dns-tunnel-ctf.pcapng | DNS Tunneling | Abnormally long DNS queries, C2 via DNS |
| wifi-ctf.pcapng | WiFi Analysis | 802.11 frames, deauth attacks, handshakes |
| ctf-challenge.pcapng | Mixed | General CTF challenge traffic |
